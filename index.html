"The Problem?"



Security provided by complexity

Quantum computers and AI will render current encryptions obsolete. Conventional solutions to this problem only contemplate increasing computational complexity and load; a solution we find to be logically inefficient and, ultimately, insufficient. This process will slow computer processing and data transmission speeds as well as consume excessive amounts of electricity. Alternative methods like using radioactive isotope decay to supply a growing market with pseudorandom number sets are logically inefficient; the costs are relatively large, and the systems don’t generate enough volume.

Ideal Solution

What does a perfect solution look like?

The perfect solution would consist of an evergreen global network of random variables that can easily and inexpensively supply a global computational network. We have identified a potentially unpredictable and non-factorable, but highly non-computational source for security keys that have minimal computational load. Our source has the potential to supply an unlimited number of keys.

First Steps

Where does the road to perfection begin?

We have already collected data sets for internal testing, which has indicated true randomness in our data. We will now seek external testing and validation. We will collect additional samples, using different techniques for each sample set, analyze them and identify the random values. We will submit these datasets to cryptography experts in academic, government, and private institutions. We anticipate this challenge will be accepted by smaller institutions before scaling to larger, more esteemed organizations.
