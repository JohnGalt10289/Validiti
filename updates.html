Data testing and analysis

Data testing is a two part adventure. First we need to runa series of collection samples using different techniques for each sample set. Second we take the data, analyse it and identify the random values to be submitted for review by the crytology specialists.

Challenge Competition

Set up data collection and analysis programs with institutions

The majority of these will be held at colleges with both social science and cryptography departments. We will use each of these departments to both collect and test the data against the most current scientific standards. This will have to be repeated many times before a standard for randomness can be established.
